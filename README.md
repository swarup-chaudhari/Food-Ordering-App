# 🍔 Food Ordering System

A full-stack Food Ordering System built using **React.js**, **Spring Boot**, and **MySQL**. The application enables users to browse food items, place orders, and track their order history, while administrators can efficiently manage menu items, categories, users, and orders through a dedicated admin dashboard.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- JWT Authentication
- Browse Food Menu
- Search Food Items
- View Food Details
- Add Items to Cart
- Place Orders
- Order History
- Responsive User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Food-Ordering-App.git
```

### Backend Setup

1. Open the backend project in IntelliJ IDEA or Eclipse.
2. Configure the MySQL database in `application.properties`.
3. Run the Spring Boot application.

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

The React application will run at:

```
http://localhost:3000
```

The Spring Boot backend will run at:

```
http://localhost:8080
```
